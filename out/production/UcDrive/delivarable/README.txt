Distributed Systems: Project Part I, BSc in Informatics Engineering, FCTUC, 2021-2022

Code testing:

To test the code, complete the following steps:


1- Run the bash file project_config.sh . This script will create the necessary directories and files.

2- Open a terminal in the 'Client' folder and type the command:

java -jar Client.jar

3- Open a terminal in the 'PrimaryServer' folder and type the command:

java -jar PrimaryServer.jar

4- Open a terminal in the 'PrimaryServer' folder and type the command:

java -jar SecondaryServer.jar

5- In the client application, in order to access the application features, prompt 'n' when asked whether the server configuration should be altered

6- Insert the 'help' command to know what commands are available.
